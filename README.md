# Big Bash League




### Installation
1. Clone the repository to your local machine:

 ```bash
   git clone https://github.com/SunnyArora7/big_bash_league.git
 ```

2. Navigate to the project directory:

 ```bash
   cd big_bash_league
 ```

3. Install dependencies by running this command in IDE terminal:

 ```bash
   flutter pub get
 ```
